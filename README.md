### Instrucciones

1. Crea una Skin llamada "ixtelolotli modo claro" + el texto que quieras. (Recuerda agregar algo más como tu usuario)
2. Copia lo que hay en `light.css`
3. En modo avanzado marca las opciones "parent only" y "prefers-color-scheme: light"
4. Haz los mismos pasos para el modo oscuro con el archivo `night.css` pero con "prefers-color-scheme: dark"
5. Crea otra skin para el modo celular, pero en opciones avanzadas selecciona "parent only" y "only screen and (max-width:42em)"
6. Por ultimo crea otra skin para la base, no es importante lo que este ahí pero recomiendo poner el tamaño de la fuente así puedes decidir que tamaño usar: 
   
```css
#workskin p {
  font-size: 100%;
}
```
  En las opciones de "parent skin" colocaremos todos los skins que creamos. Y por último recomiendo usar los [AO3 Icons 2.0 por ZerafinaCSS](https://archiveofourown.org/works/57331222) y [ Stats Icons with Hover Text por ZerafinaCSS](https://archiveofourown.org/works/55604875) y listo! Solo queda usar la skin base.

### English

### Instructions

1. Create a new skin named "ixtelolotli light mode" (I recommend adding something unique, like your username).
2. Copy the contents of the `light.css` file into the CSS box.
3. In the advanced options, check "parent only" and add "prefers-color-scheme: light".
4. Repeat these steps for the dark mode using the `night.css` file, but this time with "prefers-color-scheme: dark".
5. Create an additional skin for mobile devices. In the advanced options, check "parent only" and add "only screen and (max-width:42em)".
6. Finally, create a base skin. It doesn't need to have much code, but I highly recommend setting your font properties here so you can easily adjust the size: 
   
```css
#workskin p {
  font-size: 100%;
}
```

In the "parent skins" section, add all the skins you created in the previous steps. Lastly, I highly recommend using the [AO3 Icons 2.0 by ZerafinaCSS](https://archiveofourown.org/works/57331222) and [Stats Icons with Hover Text by ZerafinaCSS](https://archiveofourown.org/works/55604875). And that's it\! Now you just need to use your new base skin.
